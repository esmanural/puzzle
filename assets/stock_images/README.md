# Stock Images Folder

This folder contains ready-to-use images for the Jigsaw Puzzle Game.

## How to Use

1. Add images (.png, .jpg, .jpeg, or .bmp) to this folder
2. Start the game
3. In the image selection screen, you'll see your images in the "Stock Images" section
4. Click on any image to quickly start the game

## Recommendations

- **Minimum size**: 400x400 pixels
- **Maximum size**: 2000x2000 pixels
- **Best images**: Clear and colorful images provide better puzzle experience
- **Suggested types**: Landscapes, animals, artwork, geometric patterns, cityscapes

## Example Images

Good puzzle images include:
- Nature landscapes
- Animal photos
- Artwork and paintings
- Geometric patterns
- City skylines
- Abstract designs

## Note

If this folder is empty, only the "Upload My Image" option will appear in the game.
